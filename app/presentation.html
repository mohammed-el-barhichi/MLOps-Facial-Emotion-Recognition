Présentation du Projet : Classification d'Émotions Faciales

NB : Vous pouvez naviguer vers /docs pour tester les routes de notre API. Ci-dessous la présentation du projet

Ce projet vise à classer l'émotion affichée sur le visage d'une personne dans l'une des sept catégories prédéfinies à l'aide de réseaux neuronaux convolutionnels profonds...

Méthodologie de l'Algorithme de Classification

L'algorithme commence par utiliser la méthode de cascade de Haar pour détecter les visages dans chaque image provenant de la webcam...

Intégration Web pour une Interaction Conviviale

Notons également que dans le cadre de ce projet, nous avons envisagé l'utilisation d'une page HTML comportant deux cases...

Explication des Endpoints dans le Fichier Principal du Serveur :

  • Prédire l'Émotion depuis une Photo :
    • Type : GET
    • Fonction : Cet endpoint déclenche la fonction predict_from_taken_photo lorsque le client envoie une requête GET. Cette fonction utilise la caméra du dispositif pour prendre une photo et prédire l'émotion associée.
    • Exemple :
      • GET /predict_from_taken_photo
  • Prédire l'Émotion depuis une Image Téléversée :
    • Type : POST
    • Fonction : Lorsqu'un client envoie une requête POST à cette route avec un fichier image en pièce jointe, la fonction predict_from_uploaded_image est appelée. Cette fonction stocke l'image dans le répertoire "uploaded_images" et prédit l'émotion associée.
    • Exemple :
      • POST /predict_from_uploaded_image
  • Métriques du Machine Learning :
    • Type : GET
    • Fonction : Cet endpoint évalue les métriques d'apprentissage automatique pour le modèle et met à jour les métriques concernées.
    • Exemple :
      • GET /metrics_machine_learning
  • / - Page d'Accueil :
    • Type : GET
    • Fonction : Cette route renvoie un message de bienvenue lorsque le client accède à la racine de l'application. Elle sert de page d'accueil simple.
    • Exemple :
      • GET /

Ces routes définissent les fonctionnalités de base de l'application FastAPI, allant de la prise de photos à la prédiction d'émotion en passant par le téléversement d'images, avec une page d'accueil standard lorsqu'on accède à la racine de l'application.
